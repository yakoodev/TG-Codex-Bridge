# C2 — Личка: `/mkproj` проводник по файловой системе (все диски)

## Цель
Дать UI для выбора директории кликами, с пагинацией и поиском.

## Сделать
- Одна “сессия выбора” на пользователя:
  - одно сообщение, редактируемое (inline keyboard)
- Листинг:
  - текущая директория
  - кнопки папок (только директории)
  - `Up`, `Prev/Next`, `Search`, `Enter path`, `Select`
- Ограничения:
  - `PAGE_SIZE=30`, max элементов 300
  - скрывать: `.git`, `node_modules`, `bin`, `obj`, `Library`, `Temp`
- “Все диски”:
  - На Windows внутри контейнера это tricky; в Linux-контейнере диски хоста должны быть смонтированы (см. A2).
  - Корень проводника: `/host` (куда смонтированы диски/пути).
- Валидация выбранного пути: существует, директория, доступна.

## Критерии приёмки
- UI не спамит сообщениями, а редактирует одно.
- Можно найти папку через поиск/ввод пути и выбрать.
- Выбор возвращает подтверждение и запускает создание topic (см. C3).

## Как проверить
- В личке `/mkproj` → пройти по папкам → выбрать → увидеть “Создаю проект…”.
