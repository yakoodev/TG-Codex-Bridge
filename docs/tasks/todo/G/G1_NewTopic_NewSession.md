# G1 - `/new`: новый topic + новая Codex-сессия

## Status
Done.

## Что сделано
- Команда `/new` в topic:
  - создаёт новый Telegram topic в той же группе;
  - создаёт новую запись `topics` с тем же `project_id`;
  - запускает в новом topic Codex с промптом `/new`, чтобы инициализировать новую сессию.
- Старый topic продолжает работать со своим текущим контекстом.
- Новый topic стартует как отдельный контекст.

## Проверка
- В старом topic можно продолжать диалог без изменений.
- После `/new` создаётся новый thread, и ответы в нём идут через новую сессию.
