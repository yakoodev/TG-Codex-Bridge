# F2 — Парсинг `NN% context left` из вывода + обновление title

## Цель
Подхватывать “остаток контекста” и отражать в названии темы.

## Сделать
- Regex: `(?i)\b(\d{1,3})\s*%\s*context\s+left\b`
- Сохранять последнее значение за job в `topics.context_left_percent`
- При обновлении — прокидывать в title.

## Критерии приёмки
- Если строка встречалась — % обновлён.
- Если не встречалась — остаётся `??%`.

## Как проверить
- Сымитировать вывод с `72% context left` → увидеть 72% в title.
